Vim Clojure Conceal
===================

Unicode lambdas and such for clojure using Vim's conceal feature


Installation
============

Drop clojure.vim to `~/.vim/after/syntax` (Linux/Mac OS X/BSD) or
`~/vimfiles/after/syntax` folder (Windows). Or use pathogen or vundle.
